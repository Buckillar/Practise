***Color Selector*** - Added 09/03/2024

Handy tool to create color scheme and then be able to use in lots of media such as Office Tools

Simple web app where you can add table elements like this:

      <tr>
        <td>
          <p> Name of color </p>
          <p> Hex Code of color </p>
          <p> RGB or other information </p>
        </td>
        <td>
          <p>***</p>
          <p>***</p>
          <p>***</p>
        </td>
        <td>
          <p>***</p>
          <p>***</p>
          <p>***</p>
        </td>
      </tr>

The page will auto exchange the hex code to make it a background color and also click to copy to clipboard
